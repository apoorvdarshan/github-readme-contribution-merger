GitHub Contribution Merger

Merge contribution graphs from multiple GitHub users into a single SVG heatmap.

Usage /api/merge?users=torvalds,gvanrossum
Overlay mode /api/merge?users=torvalds,gvanrossum&mode=overlay&theme=github-dark

Params: users (required), mode (sum|overlay), theme (github|github-dark|blue|purple|orange)
